# CryptoReward · 点赞赢加密货币

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Ferin94ye-a11y%2Fcryptoreward)

一个暗黑风格的加密货币奖励平台。用户点赞累积积分，积分可兑换 BTC、ETH、USDT 等主流加密货币。

## 功能

- 点赞累计积分（每日限 50 次，24 小时重置）
- 实时统计总点赞数、参与人数、总积分
- 排行榜显示活跃用户
- 后台管理面板（管理员登录后可查看数据、审核兑换）
